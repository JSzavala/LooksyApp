# looksy-app

Estructura del proyecto Looksly, una aplicación de moda construida con Kotlin Multiplatform y Compose Multiplatform.

## 🚀 Tecnologías principales

- **Kotlin Multiplatform** - Código compartido entre Android e iOS
- **Compose Multiplatform** - UI compartida
- **Koin** - Inyección de dependencias
- **SQLDelight** - Base de datos local
- **Ktor** - Cliente HTTP
- **DataStore** - Almacenamiento de preferencias

## 📱 Features principales

- Feed de productos y outfits
- Búsqueda y filtros avanzados
- Perfiles de usuario y tiendas
- Carrito de compras
- Panel para proveedores
- Autenticación biométrica
- Mapas y ubicaciones
